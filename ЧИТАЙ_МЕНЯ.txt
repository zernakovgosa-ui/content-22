  TREZZY CONTENT FACTORY — КАК ЗАПУСТИТЬ (читай 1 минуту)
================================================================

ШАГ 1. РАСПАКОВКА (важно!)
--------------------------------
Распакуй архив так, чтобы файлы лежали ПРЯМО здесь:

   C:\trezzy-content-factory\START.bat
   C:\trezzy-content-factory\apps\
   C:\trezzy-content-factory\packages\

А НЕ так (это ошибка — лишняя папка внутри):

   C:\trezzy-content-factory\trezzy-content-factory-FIXED\START.bat   <- НЕПРАВИЛЬНО

Если после распаковки видишь внутри ещё одну папку с тем же именем —
зайди в неё, выдели всё (Ctrl+A) и перенеси на уровень выше.

Проверка: файл START.bat должен лежать рядом с папкой "apps".


ШАГ 2. ЗАПУСК (один клик)
--------------------------------
Просто дважды кликни:  START.bat

Первый раз он 2-5 минут поставит всё нужное (Python-окружение).
Дальше будет запускаться за секунды.
Откроются 2 чёрных окна (это сервер — НЕ закрывай их пока работаешь)
и сам откроется браузер с дашбордом.

Нужен Python 3.11+ (один раз поставить с python.org,
обязательно галка "Add Python to PATH").


ШАГ 3. БЫСТРЫЙ РОЛИК (работает сразу, без ключей)
--------------------------------
В дашборде:  Создать видео -> Режим рендера = fast -> Создать видео.
Через ~15 сек готовый ролик появится в:
   C:\trezzy-content-factory\output\latest\final.mp4


ШАГ 4. ГОВОРЯЩИЙ АВАТАР (нужен бесплатный ключ + фото)
--------------------------------
4.1. Положи ОДНО фото героя (одет, лицо+плечи, ровный фон) в папку:
        C:\trezzy-content-factory\assets\avatar\
     и назови файл:  trezzy_face.png

4.2. Получи бесплатный ключ на https://www.d-id.com/
        Зарегайся -> Settings -> API Keys -> скопируй ключ.

4.3. В дашборде:  Настройки -> блок "Говорящий аватар (D-ID)":
        - D-ID API key      -> вставь ключ
        - D-ID avatar image -> trezzy_face.png   (это уже вписано)
        - D-ID voice id      -> ru-RU-SvetlanaNeural  (уже вписано)
        Нажми "Сохранить".
     ВАЖНО: ключ и имя картинки вписываешь ЗДЕСЬ, в дашборде.
     Файл data\settings.json руками открывать НЕ нужно.

4.4. Создать видео -> Режим рендера = avatar -> Создать видео.

Если D-ID выдаст ошибку модерации (451) — значит фото не прошло
(слишком откровенное / в очках). Дай фото анфас, одет, без очков.
