Сюда положи ОДНУ картинку героя для говорящего аватара (режим avatar).

Требования к фото (иначе D-ID отклонит по модерации):
  - лицо + плечи, анфас, смотрит в камеру
  - человек ОДЕТ (без купальника/белья)
  - чистый ровный фон, хорошее освещение
  - желательно без тёмных очков (мешают синхронизации губ/глаз)
  - png или jpg

После того как положишь файл сюда (например trezzy_face.png):
  1. Открой дашборд → Настройки
  2. Впиши имя файла в поле "D-ID avatar image": trezzy_face.png
  3. Впиши свой D-ID API key
  4. Сохрани
