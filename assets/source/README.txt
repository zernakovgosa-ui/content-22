TREZZY Content Factory — папка исходников для режима CLIP
=========================================================

Сюда кладёшь ДЛИННЫЕ видео (фильм, сериал, подкаст, своё длинное видео),
из которых «завод» сам нарежет короткие вертикальные клипы.

Как пользоваться:
  1. Скопируй сюда видеофайл, например:  movie.mp4
  2. В дашборде выбери «Режим рендера» → clip
  3. В поле «Исходное видео» впиши имя файла:  movie.mp4
  4. Укажи «Сколько клипов» и нажми «Создать видео».

Что произойдёт автоматически:
  • речь распознаётся (Groq Whisper, бесплатный тариф — ключ в Настройках);
  • AI выбирает самые «залётные» моменты;
  • каждый момент режется в 9:16 с фокусом на лицо;
  • вшиваются субтитры по озвучке;
  • готовые клипы появляются в output/jobs/<id>/clip_01.mp4, clip_02.mp4 …

Поддерживаются обычные форматы: .mp4, .mov, .mkv, .webm, .avi
Авторские права: используй своё/лицензионное видео. Чужие фильмы/сериалы
при заливе в соцсети ловят копирайт-страйки.
