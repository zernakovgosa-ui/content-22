Добавить аккаунт

🔥 Мини-прогрев нового канала (рекомендация, НЕ блокирует — но ускоряет первый залёт и поднимает цену канала при продаже):
1) Дай каналу 2–5 дней «отлёжки» · 2) Оформи: аватар, шапка, описание · 3) Включи расширенные функции в YouTube Studio (свежий канал без них часто сидит на 0–300 просмотров) · 4) Немного живой активности в нише. Хочешь придержать канал на это время — нажми «🔥 прогрет», переключится в «🧊» и посты на него планироваться не будут, пока не вернёшь.
Google Cloud Console → APIs & Services → включи «YouTube Data API v3» → Credentials → OAuth client ID → тип Web application (Веб-приложение). В Authorized redirect URIs добавь ТОЧНО: http://localhost:8002/auth/yt/callback (http, порт 8002, без слэша в конце). Один клиент на все каналы. Consent screen: добавь себя в Test users или переведи в «In production» (в Testing токен живёт 7 дней). После добавления аккаунта нажми «🔗 Подключить» и войди в Google-аккаунт ЭТОГО канала.
